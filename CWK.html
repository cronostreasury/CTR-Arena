How Scoring Works

💰

Buy Volume

3x

📤

1x

🛡️

Anti-Wash

0-1x

Sells approx Buys -> score -> zero. Net buyers win.

Scoring Formula

Repeated buy-sell without growing position = score -> 0. Only real inflow earns max rewards.

// Step 1: Raw Points Raw Points = (3 x Buy Volume) + (1 x Sell Volume) // Step 2: Anti-Wash Factor Anti-Wash = max( min( (Buys - Sells) / Buys, 1 ), 0 ) // Step 3: Holding Bonus (max +50%) Holding Bonus = 1 + (Hold Duration / Season Length) x 0.5 // FINAL SCORE Final Score = Raw Points x Anti-Wash x Holding Bonus
